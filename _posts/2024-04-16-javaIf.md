# Java 조건문

## if문 
- If - >, < == : 연속 데이터 범위 비교 : if() esle if() ….
1. IF 문
	if (조건) ~ ;
	If (조건) ; ~
	- 둘다 오류는 없지만 첫번째 문장은 " 조건 > ~ 처리 ; " 하는데 
	두번째 문장은 " 조건 > ; " 이기 때문에 처리문이 실행되지 않는다. 찾기 어려움.
	
	If (조건) ; else ~;  -> 조건에 만족하면 처리문 없음, 만족하지 않으면 처리문 실행.
	If (!조건) ~ ;   -> 조건에 만족하지 않으면 실행한다.
	
	A. 중첩 if 문
	- If (조건) { 
		if (조건) ~ 
		else ~ ;
	}
	
	- If (조건) {
		~ ;
	} else If (조건) {
			~;
	}
